Instructions for running the app:
1. Go to Qualcomm's AI Hub
2. Download LaMa-Dilated (tflite)
3. Download AOT-GAN (tflite)
4. Download the model at the following link: https://drive.google.com/drive/folders/1_BiHUiOvvVpJHvf4nOpN028ujW0aNssR?usp=sharing
5. Put the downloaded models in final-codes-26_world_domination/App/android/ImageInpainting/src/main/assets/
6. Open the final-codes-26_world_domination/App/android folder in Android Studio. You will see several projects that can be built; select
'ImageInpainting' as the app to build.
7. Run the code on the connected device.
